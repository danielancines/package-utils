# 📦 Package.Utils

A lightweight and practical utility library for .NET projects, designed to simplify common development tasks and improve productivity.

---

## ✨ Features

* 🔧 ObservableCommands and Generic Observable Command to help to implement your commands using MVVM pattern
* 🧩 Easy to integrate into any .NET project

---

## 📥 Installation

Install via NuGet:

```bash
dotnet add package DanielAncines.Utils
```

Or using Package Manager:

```powershell
Install-Package DanielAncines.Utils
```

---

## 🚀 Getting Started

After installing the package, you can start using the utilities in your project:

Please, take a look on ** Utils.SampleProject **, that you will find a basic implementation of Observable and Generic Observable Commands
> Replace `SomeUtility` with the actual classes available in your package.
